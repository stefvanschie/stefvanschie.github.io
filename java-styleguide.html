Java Styleguide

This is a styleguide I created for java development. This is more for me as a reference, but if you like it you can of course use it too. This guide is based on the K&R style, but more detailed.

Indention

The indention of code blocks (e.g. if/else if/else, try/catch, classes, for loops, (do/)while loops, methods) is as follows:

if {
} else if {
} else {
}
