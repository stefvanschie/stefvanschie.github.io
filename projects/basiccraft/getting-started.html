Projects - Basic Craft - Getting Started

To start using Basic Craft you have to drop the Basic Craft file into your plugins folder. Then start your server and a new folder called BasicCraft should have been created.

Now we have to make a new file with a .bcs extension. Although this is not mandatory it is still recommended. Once you have created your file you can continue to the next tutorial.
